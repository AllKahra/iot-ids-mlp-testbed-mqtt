# Organização Final do Projeto

Esta pasta reúne os principais artefatos utilizados no relatório final do projeto:

- ambiente Docker;
- PCAPs finais e hashes;
- CSVs extraídos pelo CICFlowMeter;
- datasets processados;
- modelos treinados;
- resultados do Blind17;
- avaliação binária, multiclasse, malicious-only e macroclasses;
- arquivos para Orange;
- scripts principais usados no experimento.

A pasta `final_artifacts/` foi criada como pacote limpo de entrega. Os diretórios originais do projeto foram preservados para manter rastreabilidade e evitar quebra de scripts.
