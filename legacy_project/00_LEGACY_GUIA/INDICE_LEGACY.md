# Índice do Legacy Project

| Pasta/arquivo | Conteúdo |
|---|---|
| `attacker/` | Scripts relacionados ao botnet-controller e geração de tráfego malicioso |
| `c2/` | Fake C2 utilizado no testbed |
| `capture/` | PCAPs gerados nas rodadas, testes cegos e reforços |
| `cicflowmeter/` | CSVs brutos e datasets processados pelo CICFlowMeter |
| `gateway/` | Gateway IoT simulado |
| `ml/` | Scripts de pré-processamento, treino, avaliação e triagem |
| `mqtt/` | Configuração do broker MQTT |
| `orange/` | Arquivos usados no Orange durante o desenvolvimento |
| `results/` | Resultados experimentais, modelos, métricas e matrizes |
| `scripts/` | Scripts de captura, conversão e apoio |
| `sensors/` | Sensores IoT simulados |
| `docker-compose.yml` | Definição do ambiente Docker original |
| `requirements.txt` | Dependências Python do projeto original |

## Observação

Nem tudo nesta pasta é resultado final. Muitos arquivos são históricos, intermediários ou experimentais.
